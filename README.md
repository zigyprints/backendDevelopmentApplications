# Backend Assignment - Prajjwal Kapoor

## Local setup

You should have installed nodejs >=v14 and npm to run this application.

```bash
  cd googledoc-clone\
  npm install
  npm build
  npm run start-prod
```

## Tech Stack

_Express, nodejs, momgodb_

## Features

- Create Single Todo
- View All Todos
- View Single Todos
- Update Single Todos
- Delete a Todo
- Error Handling Middleware
- Morgan Request Logging
